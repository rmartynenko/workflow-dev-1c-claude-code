# Команда: Начало сессии

## Назначение

Инициализация рабочей сессии с Claude Code. Загрузка контекста проекта и синхронизация состояния.

## Когда использовать

- В начале каждой новой сессии работы
- После длительного перерыва
- При переключении между проектами

## Действия

### 1. Очистка контекста

```
/clear
```

### 2. Загрузка Memory Bank

Прочитать файлы в следующем порядке:

1. **CLAUDE.md** — главная конфигурация проекта
2. **activeContext.md** — текущее состояние и фокус
3. **progress.md** — последние записи (5-10 записей)
4. **techContext.md** — если нужен технический контекст
5. **systemPatterns.md** — если работа связана с архитектурой

### 3. Синхронизация с трекером задач (если настроен)

```bash
# Для Yougile
$YOUGILE_CLI task list --column-id <in_progress_id>

# Для GitHub Issues
gh issue list --assignee @me --state open
```

### 4. Проверка состояния репозитория

```bash
git status
git log --oneline -5
```

### 5. Вывод сводки

```markdown
## Сессия начата

### Проект
[Название проекта]

### Текущий фокус
[Из activeContext.md]

### Активные задачи
- [Задача 1]
- [Задача 2]

### Последние изменения
- [Дата]: [Описание]

### Готов к работе
Чем могу помочь?
```

## Пример вызова

```
Начни новую сессию по команде .claude/commands/start-session.md
```

или короткая форма:

```
/start
```

## Автоматизация

При наличии MCP сервера задач — автоматически синхронизировать состояние.

## Чек-лист начала сессии

- [ ] Контекст очищен
- [ ] Memory Bank загружен
- [ ] Состояние git проверено
- [ ] Активные задачи известны
- [ ] Готов к работе
