| Сервер | Порт | Инструмент | Назначение |
|--------|------|------------|------------|
| 1c-docs-mcp | 8003 | docsearch | Документация платформы |
| 1c-ssl-mcp | 8008 | ssl_search | Документация БСП |
| 1c-templates-mcp | 8004 | templatesearch | Шаблоны кода |
| 1c-syntax-mcp | 8002 | syntaxcheck | Проверка синтаксиса |
| 1c-metadata-mcp | 8006 | search_metadata | Метаданные конфигурации |

### Конфигурация MCP

```json
{
  "mcpServers": {
    "1c-docs-mcp": {
      "url": "http://localhost:8003/sse",
      "transport": "sse"
    }
  }
}
```
